# Placement Management System (Java - Console Based)

## Project Description
This is a console-based Java application that manages student placements.
It allows adding students, companies, and placing eligible students
based on CGPA criteria.

## Features
- Add Student
- View Students
- Add Company
- View Companies
- Place Student based on CGPA
- View Placement Records

## Technologies Used
- Java
- ArrayList
- OOP Concepts
- Scanner (for input)

## Project Structure
- Main.java
- Student.java
- Company.java
- Placement.java

## How to Run
1. Compile all files:
javac Main.java
2. Run the program:
3. java Main
